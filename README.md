# bboxfixer

A tool for the **BBOX fiscal printer** that helps generating Windows `.bat` files so receipts and storno (cancellation) documents can be reprinted by running the generated bat files.

## Features

- Reads receipt data from **CSV** or **JSON** input files
- Generates `.bat` files containing `curl` commands that:
  - **Reprint** an existing receipt by its fiscal number
  - **Storno** (cancel/reverse) a receipt by sending a negated copy to the printer
- Configurable printer host and port
- Custom storno reason text

## Requirements

- Python 3.8+
- `curl` available on the target Windows machine

## Installation

```bash
pip install .
```

## Usage

```
bboxfixer INPUT_FILE [options]
```

### Options

| Option | Default | Description |
|---|---|---|
| `-o / --output-dir DIR` | `output` | Directory where `.bat` files are written |
| `--host HOST` | `localhost` | BBOX printer IP/hostname |
| `--port PORT` | `5618` | BBOX printer HTTP port |
| `--mode {reprint,storno,both}` | `both` | Which bat files to generate |
| `--storno-reason REASON` | `Cancellation` | Reason text on storno documents |

### Examples

Generate both reprint and storno bat files for all receipts in a CSV:

```bash
bboxfixer receipts.csv -o ./bat_files --host 192.168.1.100
```

Generate only reprint bat files using a JSON input:

```bash
bboxfixer receipts.json -o ./reprints --mode reprint
```

Generate storno bat files with a custom reason:

```bash
bboxfixer receipts.csv --mode storno --storno-reason "Customer return"
```

## Input File Format

### CSV

The CSV file must contain the following columns (in any order):

```
receipt_number,date,cashier,payment_method,item_name,quantity,unit_price,tax_rate
R001,2024-01-15,John Doe,cash,Coffee,2,1.50,27
R001,2024-01-15,John Doe,cash,Sandwich,1,3.00,5
R002,2024-01-16,Jane Smith,card,Tea,3,1.00,27
```

Multiple rows with the same `receipt_number` are merged into a single receipt.

### JSON

The JSON file must be an array of receipt objects:

```json
[
  {
    "receipt_number": "R001",
    "date": "2024-01-15",
    "cashier": "John Doe",
    "payment_method": "cash",
    "items": [
      {"name": "Coffee", "quantity": "2", "unit_price": "1.50", "tax_rate": "27"},
      {"name": "Sandwich", "quantity": "1", "unit_price": "3.00", "tax_rate": "5"}
    ]
  }
]
```

## Generated Bat File Examples

**Reprint (`reprint_R001.bat`):**

```bat
@echo off
REM Generated by bboxfixer
REM Reprint receipt R001
curl -s -X POST "http://192.168.1.100:5618/peri/print/reprint?FN=R001" ^
  -o NUL
if %errorlevel% neq 0 (
    echo ERROR: reprint failed for receipt R001
    exit /b %errorlevel%
)
echo Reprint sent for receipt R001
echo Done.
```

**Storno (`storno_R001.bat`):**

```bat
@echo off
REM Generated by bboxfixer
REM Storno for receipt R001
curl -s -X POST "http://192.168.1.100:5618/peri/print" ^
  -H "Content-Type: application/json" ^
  -d "{\"uniqueSaleNumber\":\"STORNO-R001\",...}" ^
  -o NUL
if %errorlevel% neq 0 (
    echo ERROR: print command failed
    exit /b %errorlevel%
)
echo Print command sent
echo Done.
```

## Tax Groups

The tool maps Hungarian VAT percentages to BBOX tax group codes:

| Rate | Tax Group |
|---|---|
| 27 % | 1 |
| 18 % | 4 |
| 5 % | 2 |
| 0 % | 3 |
